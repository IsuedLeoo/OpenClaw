# OpenClaw Download Instructions

OpenClaw is installed like any other app from a website. You do not need to use code or commands.

## Downloading OpenClaw

1. Visit the OpenClaw website or download page.
2. Find the version for your computer:
   - macOS users: download the `.dmg` or `.pkg` file
   - Windows users: download the `.msi` or `.exe` file
3. Click the download button and wait for the file to finish downloading.

## Installing on macOS

1. Open the downloaded `.dmg` or `.pkg` file.
2. If you opened a `.dmg`, drag the `OpenClaw.app` icon into the `Applications` folder.
3. If you opened a `.pkg`, follow the installer steps on screen.
4. Open OpenClaw from Launchpad, Spotlight, or the Applications folder.

## Installing on Windows

1. Open the downloaded `.msi` or `.exe` file.
2. Follow the setup wizard instructions.
3. When installation is complete, open OpenClaw from the Start menu or desktop shortcut.

## If you download a ZIP file

- macOS: unzip the file, then move `OpenClaw.app` to the `Applications` folder.
- Windows: unzip the file, then double-click `OpenClaw.exe` to run it.

## Starting OpenClaw

- On macOS: click the app icon in Launchpad or Finder.
- On Windows: click the app icon in the Start menu or on the desktop.

## Notes for users

- No coding or command-line steps are needed.
- If your browser asks, choose `Keep` or `Allow` for the downloaded installer.
- If your computer warns about unknown software, confirm that you downloaded from the official OpenClaw website.
